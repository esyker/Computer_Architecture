Laboratório 0
Os alunos que não tenham tido contacto com o ambiente de projecto VIVADO da Xilinx devem realizar a título de treino (com os seus próprios meios) este trabalho de laboratório (lab3 de Sistemas Digitais do MEEC - Prof. Nuno Roma) onde é feita uma introdução ao VIVADO e ao VHDL.
Para realizar o trabalho descarregue os ficheiros:
  • Enunciado do lab3 de SD (MEEC16/17)
  • Ficheiros auxiliares de projecto
  • Folha de respostas
Para realizar este trabalho sugere-se ainda:
  • o estudo do conjunto de slides 12 e 14 de Sistemas Digitais
  • a instalação do software VIVADO
  • a consulta do Guia de Instalação do Vivado Design Suite WebPack.
